# cisco-support CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a CLI for querying Cisco Support APIs (bugs, cases, EoX, PSIRT, product, software, serial, RMA) following the established patterns from cisco-axl, cisco-dime, and cisco-ise.

**Architecture:** Commander.js CLI with domain-grouped commands. OAuth 2.0 client credentials flow for Cisco API auth. Token caching with auto-refresh. Single config file (no clusters — one API endpoint). ss-cli placeholder support. Same formatter/output/audit patterns as sibling CLIs.

**Tech Stack:** Node.js, Commander.js, axios, cli-table3, @toon-format/toon, csv-stringify

```js
const assert = require("assert");
const fs = require("fs");
const path = require("path");
const os = require("os");

// Use temp dir for test isolation
const testDir = fs.mkdtempSync(path.join(os.tmpdir(), "cisco-support-test-"));
process.env.CISCO_SUPPORT_CONFIG_DIR = testDir;

const config = require("../../cli/utils/config.js");

describe("config", () => {
  it("returns empty config when no file exists", () => {
    const data = config.loadConfig();
    assert.strictEqual(data.clientId, undefined);
  });

  it("saves and loads config", () => {
    config.saveConfig({ clientId: "test-id", clientSecret: "test-secret" });
    const data = config.loadConfig();
    assert.strictEqual(data.clientId, "test-id");
    assert.strictEqual(data.clientSecret, "test-secret");
  });

  it("detects ss-cli placeholders", () => {
    assert.strictEqual(config.hasSsPlaceholders({ clientId: "<ss:123:id>" }), true);
    assert.strictEqual(config.hasSsPlaceholders({ clientId: "plain" }), false);
  });

  it("manages enabled APIs", () => {
    config.saveConfig({ clientId: "x", clientSecret: "y", enabledApis: ["bug", "psirt"] });
    const data = config.loadConfig();
    assert.deepStrictEqual(data.enabledApis, ["bug", "psirt"]);
  });
});

// Cleanup
fs.rmSync(testDir, { recursive: true });
```
